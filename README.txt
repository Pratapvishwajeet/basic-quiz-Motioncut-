Python Quiz Game

This repository contains a Python-based quiz game that utilizes object-oriented programming for enhanced structure and reusability. The game features:

Multiple-choice questions: Questions are presented with multiple options.
Scoring system: Tracks and displays the user's score.
User interaction: Allows users to input answers and provides feedback.
correct or incorrect: Display the correct answer if the user is wrong.
Final Score: Display the user's final score at the end of the quiz.

Key features:
User input and validation
Scoring system
feedback
